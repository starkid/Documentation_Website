Project aim:

To install Jekyll on my MacBook Air for building a blog that can be uploaded to GitHub Pages.

Time assigned

0.5hrs

Time Taken

08/02/2021 0.5hrs on 0.0

Contents

0.0

Open a Terminal.

xcode-select --install

Run the following cmd to agree to the license.

sudo xcodebuild -license

Install Homebrew.

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Install Ruby.

brew install ruby

Install Jekyll

sudo gem install jekyll

Check the version installed.

jekyll -v

Create a new Jekyll site at ./site

jekyll new site

Change into your new directory.

cd site

Build the site and make it available on a local server.

bundle exec jekyll serve
